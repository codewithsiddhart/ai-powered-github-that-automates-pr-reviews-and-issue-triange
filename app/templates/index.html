Get Started in 3 Steps

From zero to AI-automated repository in under 10 minutes.

1

Deploy to Render

Fork the repo, connect to Render, and deploy the web service + Redis. Set your environment variables: GITHUB_APP_ID, GITHUB_PRIVATE_KEY, GITHUB_WEBHOOK_SECRET, and at least GROQ_API_KEY.

2

Create a GitHub App

Go to GitHub → Settings → Developer Settings → GitHub Apps → New. Set the webhook URL to your Render deployment. Enable events: Pull Requests, Issues, Issue Comments, Pushes, Check Runs.

3

Install on Your Repos

Install the GitHub App on any repository. The bot starts working immediately — type /fix on any issue or /explain on any PR to see it in action.

Required Environment Variables

GITHUB_APP_ID Your GitHub App's numeric ID
GITHUB_PRIVATE_KEY RSA private key from GitHub App settings
GITHUB_WEBHOOK_SECRET 32+ char random secret for HMAC verification
GROQ_API_KEY Free at console.groq.com — primary AI provider
REDIS_URL Auto-set by Render Redis service
GEMINI_API_KEY Optional — fallback at aistudio.google.com
